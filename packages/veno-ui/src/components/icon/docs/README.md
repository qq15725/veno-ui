## 使用 mdi class 图标

安装依赖

```sh
$ npm i -D @mdi/font
```

引入 `@mdi/font/css/materialdesignicons.css`

```ts
// src/plugins/veno-ui.ts
import '@mdi/font/css/materialdesignicons.css'
```
