# 预设尺寸

尺寸有 `x-small` - 特小、`small` - 小、`medium` - 中等、`large` - 大、`x-large` - 特大。

```html
<ve-spacer gutter="md" align="center" class="bg-grey-300 p-3">
  <ve-paper size="xs" />
  <ve-paper size="sm" />
  <ve-paper size="md" />
  <ve-paper size="lg" />
  <ve-paper size="xl" />
</ve-spacer>
```
