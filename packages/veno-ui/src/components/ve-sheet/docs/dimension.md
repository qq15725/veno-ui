# 自定义尺寸

通过 `size`、`width`、`height`、`min-width`、`min-height`、`max-width`、`max-height` 自定义尺寸。

```html
<ve-row>
  <ve-col cols="auto">
    <ve-sheet size="50" />
  </ve-col>
  
  <ve-col cols="auto">
    <ve-sheet width="50" height="25" />
  </ve-col>
  
  <ve-col cols="auto">
    <ve-sheet width="25" height="50" />
  </ve-col>
</ve-row>
```
